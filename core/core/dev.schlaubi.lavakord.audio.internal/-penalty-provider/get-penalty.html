abstract fun getPenalty(node: Node, guildId: Long): Int

This method allows for adding custom penalties to {@link LavalinkSocket nodes}, making it possible to change how the node selection system works on a per-guild per-node basis. By using the provided {@link LavalinkLoadBalancer.Penalties Penalties} class you can fetch default penalties like CPU or Players.

Return

total penalty to add to this node.
